A presentation on setting up the editorial suite (software, cloning) at the conference venue can be found [here](https://www.jacow.org/Editors/SoftwareToolsandRequirementsSetupandInstallationOptions).

## Setup Strategy

- **Small conferences (4-6 editors):** Consider setting up workstations individually
- **Large conferences:** Create a master setup and clone to other computers

## General Requirements

- **Language:** All software must be installed with at least English language support
- **Version policy:** Use "latest stable version available" unless otherwise specified
- **Critical:** Ensure all patches are installed

---

## Windows PCs

### Operating System
- **Windows 10 64-bit (English)**

### Office Suites

| Software | Version | Notes |
|----------|---------|-------|
| Microsoft Office | 2019/2021 | 2021 preferred to prevent incompatibilities with Office 365 files. All patches required |
| LibreOffice | 7.4.7/7.5.4 | Alternative office suite |

### Web Browsers
- Firefox
- Google Chrome

### PDF and Document Processing

| Software | Purpose | Licensing |
|----------|---------|-----------|
| Adobe Acrobat Professional DC | PDF editing/creation | License provided by conference (patched required) |
| JACoW Acrobat configuration | JACoW-specific setup | Required configuration |
| JACoW JobOptions (JACoW-11) | Distiller settings | Must be properly installed |
| Enfocus Pitstop Professional 2022 | PDF preflight/editing | 30-day trial (activate only before conference) |

### Text Editors
- **Notepad++** (primary recommendation)
- WinEdt (30-day trial)
- EditPlus
- Gvim for Windows

### Utilities and Tools

| Category | Software | Purpose |
|----------|----------|---------|
| **Archive** | 7Zip | File compression/extraction |
| **PDF Viewing** | Ghostview, GSView, GhostScript | PDF/PostScript viewing |
| **Remote Access** | PuTTY | SSH client |
| **Security** | Anti-Virus software | System protection |
| **Image Viewing** | IrfanView, XnView | Image viewing/basic editing |
| **File Management** | DoubleCommander (OpenSource) or SpeedCommander (30-day trial) | Advanced file management |

### LaTeX Environment

| Component | Details |
|-----------|---------|
| **MiKTeX** | User Mode install (not Admin install) |
| **JACoW LaTeX classfile** | Properly installed |
| **TeX GUI** | TeXStudio (first choice) + TeXnicCenter/TeXmaker |

### Media and Graphics

| Software | Purpose |
|----------|---------|
| **VLC** | Media player |
| **GIMP** | Advanced image editing |
| **Paint.NET** | Basic image editing |
| **Handbrake** | Video transcoding |
| **VirtualDub** | Video editing |
| **PPspliT** | Presentation splitting |

### Development Tools
- **Meld** (diff/merge tool)

## Important Notes

1. **JACoW-specific configurations** must be properly installed for Acrobat and LaTeX environments
2. **Trial software** should only be activated immediately before the conference to maximize usage period
3. **Patch management** is critical for all installed software
4. **Cross-platform consistency** should be maintained where possible for editor mobility
